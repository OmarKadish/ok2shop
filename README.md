# ok2shop Web Application
## Description

*OK2Shop* is an online shopping platform that allows users to browse and purchase technology accessories such as headphones, keyboards, computer mice, and mobile devices.

## Requirements for the Project

### Requirements

* The project is written in PHP an object-oriented language.
* It's all self-written, there are no existing frameworks or libraries.
* For more security, I used (hashed passwords, protection against SQL injection, etc)
* I used MySQL database to store admin account and all the data and managed the database with PHPMyAdmin.

### Specifications:

#### The Frontend

* HTML, CSS, and JavaScript code were integrated into PHP files for Dynamic paging in the site.
* Homepage: It shows all the products.
* About page: Includes a brief bio of the site.
* Wiki page: It has info on how to use the site and a contact form for any suggestions, complaints or messages (TODO).

#### The Backend

* List of all products
* Possibility to add a register and login.
* Possibility to manage users by the admins.
* Possibility to add products to the cart and checkout.
* Logout feature for all users.
